# Week 2 Module - Searching & Sorting in C

## Overview
This repository contains the solutions and implementations of fundamental **Searching** and **Sorting** algorithms in C programming as part of the **Placement & Training Program - Week 2 Module**.

The objective of this module is to understand the working, implementation, and time complexity of commonly used searching and sorting techniques.

---

## Programs Included

### 1. Find Student Roll Number (Linear Search)
- Searches for a roll number in an unsorted array.
- Time Complexity: **O(n)**

### 2. Search Product ID (Binary Search)
- Searches for a product ID in a sorted array.
- Time Complexity: **O(log n)**

### 3. Arrange Exam Scores (Bubble Sort)
- Sorts exam scores in ascending order.
- Time Complexity: **O(n²)**

### 4. Arrange Library Book Numbers (Insertion Sort)
- Sorts book numbers efficiently for small datasets.
- Time Complexity: **O(n²)**

### 5. Rank Players by Score (Selection Sort)
- Sorts player scores by repeatedly selecting the minimum element.
- Time Complexity: **O(n²)**
